定义

一个用python开发的进行管理软件

安装

yum install supervisord.noarch

修改配置

​ cd /etc/supervisord.d

新建一个需要托管的文件

​vim cloud-admin.ini

[program:kce]
command     = /home/web_server/files/cloud-admin --config=/home/web_server/files/admin_config.toml
startsecs   = 3
redirect_stderr         = true
stdout_logfile_maxbytes = 50MB
stdout_logfile_backups  = 10
stdout_logfile          = /data/logs/kce/stdout.log
stderr_logfile = /data/logs/kce/stderr.log
autorestart = true
startretries = 3600

将command修改成自己的目录（二进制运行文件和.toml配置文件），logfile的路径改成自己的log路径

启动

​ systemctl start supervisord.service

查看状态

​ supervisorctl status all

启动服务

​ supervisorctl start 服务名

重启服务

​ supervisorctl restart 服务名

停止服务

​ supervisorctl stop 服务名
