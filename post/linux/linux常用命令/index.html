解压

filename**.tar.gz**的解压:

tar -zxvf filename.tar.gz

其中zxvf含义分别如下

z: 　　gzip 　　　　　　　 压缩格式

x: 　　extract　　　　　　　　 解压

v:　　 verbose　　　　　　　　详细信息

f: 　　file(file=archieve)　　　　文件

$ ps -ef | grep cloud
root     22323 18786  0 17:32 pts/1    00:00:00 grep --color=auto cloud
web_ser+ 54612 52979  0 17:18 ?        00:00:00 /bin/bash /XXX/run.sh
web_ser+ 54648 54612  0 17:18 ?        00:00:00 ./cloud

# 注：第一个是本进程号，后面的是父进程号
# 第一个就是ps这个命令的进程
# 第二个是程序的进程，调用run.sh脚本
# 在run.sh里又执行了./cloud命令，所以它又开了一个进程，所以54648的副进程是54642
